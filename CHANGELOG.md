## [1.0.0] - 2026-03-08

### Added

- Live website previews powered by `WebContentsView`
- Minimal right-side control rail and collapsible control panel
- Phone, tablet, and desktop viewport switching
- Loading overlay for page transitions
- Visible-area and full-page screenshot capture with clipboard copy and Downloads saving
- Public repository assets including CI, release workflow, contribution docs, security policy, and issue templates
