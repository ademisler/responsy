# Code of Conduct

## Our Standard

We want Responsy to be a respectful, constructive, and technically serious project.

Examples of behavior that contribute to a positive environment:

- Being respectful in language and tone
- Giving actionable technical feedback
- Assuming good intent while discussing tradeoffs
- Sharing context clearly instead of escalating conflict

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Discriminatory or exclusionary language
- Deliberate disruption of issues, pull requests, or discussions
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the repository owner through GitHub.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, discussions, and release conversations.
